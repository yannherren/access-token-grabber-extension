Welcome!

Thank you for installing Access Token Grabber!

Here is how to use Access Token Grabber:
  1. Activate the extension by clicking on its icon.
  2. Pin the extension to your browser toolbar for quick access.
  3. Browse to your app that triggers authenticated requests. A small "1" will appear on the icon, indicating that a token has been found.
  4. Click once on the icon or press Alt+C to copy the token to your clipboard.
  5. Paste the token into Postman, cURL, or any API client to use it.
Have fun and don't forget to visit our GitHub repo!
